Revisiting Weak-to-Strong Consistency in Semi-Supervised Semantic Segmentatio

重新审视半监督语义分割中的弱到强一致性

  • 提出了UniMatch，一种统一的双流扰动方法用于半监督语义分割

  • 设计了一种双流扰动策略,充分利用预定义的图像级强扰动空间,并利用对比学习的优势获得更具区分性的特征表示 。

没太看明白到底是为什么会提升性能的，后面还需要时间再看看

UniMatch 通过统一的双流扰动框架,有效地探索了更广泛的扰动空间,并充分利用了图像级强扰动的优势,从而在多个基准测试上取得了显著的性能提升。

Fuzzy Positive Learning for Semi-supervised Semantic Segmentation

这篇论文介绍的是模糊正向学习（Fuzzy Positive Learning）

  • 现有的半监督分割方法都很依赖于伪标签，这有可能存在错误然后误导模型训练
  • FPL利用具有一定概率为真实标签的“模糊正向”标签的有意义语义，而不仅仅是单一的伪标签
  • 两个主要的组成部分：
    • 模糊正向赋值(Fuzzy Positive Assignment, FPA)自适应地为每个像素提供适当数量的模糊正向标签。
    • 模糊正向正则化(Fuzzy Positive Regularization, FPR)鼓励模型学习模糊正向标签的语义。
  • 可以轻松集成到现有的半监督分割方法中,并在Cityscapes和Pascal VOC 2012数据集上提供持续的性能提升。

Instance-specific and Model-adaptive Supervision for Semi-supervised Semantic Segmentation

这是一种机遇实例特定的模型自适应监督方法

  • 根据教师和学生模型预测之间的加权对称交并比(IoU)来评估每个未标注实例的"难度"。这种定量的难度分析捕捉了未标注实例之间的差异和训练难度。
  • 利用难度信息：
    • 根据模型的泛化能力,自适应地对每个未标注实例应用数据增强,调整扰动程度。
    • 以课程学习的方式加权每个未标注实例的无监督损失,随着模型的进化,更多关注于较难的实例。
  • 可以在不引入额外网络组件或训练损失的情况下,提高半监督语义分割的性能,在基准数据集上达到最先进的结果。

Augmentation Matters: A Simple-yet-Effective Approach to Semi-supervised Semantic Segmentation

提出了一种简单而有效的方法 AugSeg 用于半监督语义分割。（数据增强的方法）

  • AugSeg 遵循标准的教师-学生框架,主要关注数据增强来提高半监督性能,而不是集成越来越复杂的技术。
  • 作者认为现有的数据增强应该进行调整,以更好地适应半监督场景。
    • 提出了一种简化的基于强度的增强方法,选择具有均匀采样失真强度的随机数量的变换。
    • 自适应地注入标记信息,增强置信度较低的未标记样本。
  • 无需额外组件,AugSeg 就可以在半监督语义分割基准测试中达到新的最先进水平。

Conflict-Based Cross-View Consistency for Semi-Supervised Semantic Segmentation

提出了一种新的基于冲突的跨视图一致性(CCVC)方法用于半监督语义分割(SSS)。

  • 用特征差异损失来鼓励两个子网络从同一输入中学习不同的特征,同时做出一致的预测。
  • 提出了一种基于冲突的伪标签(CPL)方法,使两个子网络能够从相互冲突的预测中学习更有用的语义信息,从而实现更稳定的训练过程。

Hunting Sparsity: Density-Guided Contrastive Learning for Semi-Supervised Semantic Segmentation

提出了一种基于密度引导的对比学习(Density-Guided Contrastive Learning, DGCL)策略用于半监督语义分割。

  • 除了利用标签和伪标签信息外,还可以从特征空间的几何结构中提取有效的监督信息。
  • 特征密度来定位特征簇中的稀疏区域,因为这些稀疏区域往往比密集区域训练得更差。
  • DGCL策略将稀疏区域的锚定特征推向由高密度正样本键近似的簇中心,以增加特征空间中的类内紧凑性。
  • 设计了一个多尺度密度估计模块来获得稳健的密度估计,并将标签引导的自训练和密度引导的几何正则化统一到一个训练框架中。

没有代码

本文主要想法是利用像素之间的相关性来构建更加安全有效的监督信号，构建了一个铭文RankMatch的正则一致性框架，包括构建代表性agnet来建模像素之间的相关性，以及通过建模agent之间的关系来进一步挖掘agent的潜力，实现基于排序感知的相关性一致性。

  1. 定义教师网络 𝑓𝑇 和学生网络 𝑓𝑆,以及弱增强 𝑎𝑢𝑔(⋅) 和强增强 𝐴𝑢𝑔(⋅) 操作。
  2. 对于有标签数据,计算学生网络 𝑓𝑆 的监督损失 𝐿𝑠𝑢𝑝。
  3. 对于无标签数据,从教师网络 𝑓𝑇 获得伪标签,并计算像素一致性正则化损失 𝐿𝑟𝑒𝑔。
  4. 通过正交选择策略为 𝑓𝑇 和 𝑓𝑆 分别获得代表性 agent。
  5. 计算 agent 之间的相关性,并将其转换为 agent 排序概率分布。
  6. 计算基于排序感知的相关性一致性正则化损失 𝐿𝑟𝑎𝑛𝑘。
  7. 最终将监督损失、像素一致性正则化损失和排序感知相关性一致性正则化损失相加,进行梯度反向传播更新模型。

Towards the Uncharted:

提出了一种名为密度下降特征扰动(Density-Descending Feature Perturbation, DDFP)的新颖特征级一致性学习框架,用于半监督语义分割。其核心思想是利用特征密度信息来引导分割模型探索特征空间中较为稀疏的区域,这与半监督学习中的低密度分离假设相一致

  • 引入了一个基于归一化流的密度估计模块,用于有效地捕捉特征空间中的密度分布。这个密度估计器可以将特征映射到潜在空间,并通过变量变换公式估计原始特征空间中的密度分布。
  • 在训练分割模型的过程中,密度估计器会被优化以最大化标注和未标注特征的对数似然。这样可以利用标注数据中的知识来建立类别与高斯分量之间的联系,并引导密度估计器的初始化。同时,在未标注特征上的无监督似然最大化也使密度估计器能够自适应地调整其表示,以更好地与潜在的数据分布保持一致。
  • 在优化密度估计器的同时,利用它来生成密度下降的扰动,用于特征级一致性学习。具体来说,通过最小化特征的对数似然,找到能够导致特征密度最大下降的扰动方向。然后将这种密度下降扰动注入到原始特征中,得到密度下降版本的特征。
  • 这些密度下降的特征会被用于监督原始特征的预测结果,迫使分类器探索低密度区域,从而有效地正则化决策边界。

CorrMatch

  • CorrMatch用像素之间的相关性图来改善对未标注数据的利用
  • 两种新颖的标签传播策略：
    • 像素传播:将相关性图传播到模型的预测中,增强对成对相似性的感知。
    • 区域传播:从相关性图中提取无关类别的形状信息,准确地为高置信度区域分配类别标签。

ECCV 2024 半监督语义分割 汇总

Beyond Pixels: Semi-Supervised Semantic Segmentation with a Multi-scale Patch-based

Multi-Label Classifier

conclusion

从结论上来说，这个论文提出了一个方法，或者说一个插件（按论文的意思是可以融合的plug-in模型），叫做“基于小块的多尺度多标签分类器（Multi-scale Patch-based Multi-label Classifier(MPMC)）”，可以和优化对于小物体的检测并减小伪标签的噪声。

MPMC 提供补丁级监督，能够区分补丁内不同类别的像素区域。此外，MPMC 还能学习自适应伪标签权重，利用补丁级分类来减轻教师的伪标签超视觉噪声对学生的影响。这种轻量级模块可以集成到任何 SSS 框架中，从而显著提高其性能。

Introduction

  • 添加了一个块级别的监督
  • MPMC包含两个可以调整的学习权重来降低来自教师模型的噪声伪标签的影响。
  • 他们验证了医学上的图像，说明这个方法对非自然图像也是有用的。

把一个图像快和相应分各种的感受野r定义为：{x_rl,y_rl}l

y_{r}^{l}是一个多标签目标，其中感受野 r 中存在的类是正类，不存在的类是负类。

MPMC模块旨在通过引入基于patch的多标签分类任务来增强语义分割框架中的像索区域区分能力，并缓解教师模型传递给学生模型的伪标签噪声的影响。MPMC可以集成到教师模型g和学生模型中。

对于有标签图像，MPMC会根据patch中存在的类别对patch表示进行多标签分类。利用不同尺度的patch表示有助于捕获形状和大小方面的上下文信息，从而增强分割器的类别区分能力。

对于无标签图像，除了进行patch分类，MPWC还会计算一个伪标签权重图，以减轻教师模型的噪声伪标签对学生模型的影响。具体来说，MPMC会根据其对每个类别的预测置信度为每个patch中的伪标签分配权重，以提高伪标签过程的可靠性和准确性。

Weighting Pseudo-Labels via High-Activation Feature Index Similarity and Object Detection for Semi-Supervised Segmentation

提出了一种可以可靠地从伪标签中学习进行半监督语义分割的方法。

  • 一种基于像素表示与类原型之间相似性的伪标签加权方法,使用鲁棒于噪声的排序统计相似性
  • 通过融合训练好的目标检测模型和语义分割模型的预测结果,提出一种集成方法来识别可靠的伪标签像素。
  • 可以集成到现有的班进度分割框架中。

Space Engage: Collaborative Space Supervision for Contrastive-based Semi-Supervised Semantic Segmentation

利用逻辑和表示空间之间的协作来提高半监督语义分割的性能。

  • 利用表示中的语义信息,在无标签训练期间获得更可靠的指导,并增强逻辑和表示空间之间的知识交流。
  • 将表示和原型之间的相似性作为新的指标来指导表示空间中的学习过程,这比使用置信度作为指标更有效。

Logic-induced Diagnostic Reasoning for Semi-supervised Semantic Segmentation

看到一篇浙大的

提出了一种名为LOGICDIAG的新型神经逻辑半监督学习框架,以解决现有伪标签方法在半监督语义分割中的局限性。

  • 现有的半监督语义分割方法严重依赖于伪标签,容易受到确认偏差和错误累积的影响。
  • LOGICDIAG利用逻辑规则形式的符号知识诊断和解决伪标签中的冲突,从而提高伪标签的质量。
  • 在基准数据集上的广泛实验证明了LOGICDIAG的有效性和通用性,特别是在标签稀缺的情况下。
  • 突出了将符号推理系统地集成到神经学习方法中的有前景的机会。

比较不常见，可能需要多一点时间来理解

Large-Scale Land Cover Mapping with Fine-Grained Classes via Class-Aware Semi-Supervised Semantic Segmentation

提出了一个统一的面向类别的半监督语义分割框架,以解决大规模土地覆盖映射中细粒度类别的挑战，针对性的，不太清楚有没有帮助

CFCG: Semi-Supervised Semantic Segmentation via Cross-Fusion and Contour Guidance Supervision

提出了一种名为CFCG的新型半监督语义分割(SSSS)框架,该框架结合了交叉融合和轮廓引导监督。

  • CFCG采用图像级和特征级扰动来扩展特征分布,并推动一致性正则化的极限。
  • 交叉融合监督(CFS)机制利用多个学习器来增强伪标签的质量。
  • 自适应轮廓引导模块(ACGM)利用语义轮廓作为指导,有效地识别伪标签中不可靠的空间区域。

Enhanced Soft Label for Semi-Supervised Semantic Segmentation

提出了一种增强型软标签(ESL)框架用于半监督语义分割。

  • 现有的使用伪标签的自训练方法通常会丢弃低置信度的预测,这可能会导致信息损失。相反,论文认为这些具有高价值监督信号的难样本对于模型训练至关重要。
  • 论文引入了动态软标签(DSL)模块,动态维护高概率类别,保持标签"软"以充分利用高熵预测。
  • 为了解决DSL引入的主导类别之间的歧义问题,论文进一步提出了一种像素到部件对比学习方法,与无监督的物体部件分组机制协作。

XNet

提出了一种基于小波变换的低频和高频融合模型,称为XNet,它可以同时在完全监督和半监督生物医学图像语义分割中实现最先进的性能。

  • XNet使用小波变换生成低频(LF)和高频(HF)图像,然后分别输入到不同的编码器中提取语义和细节特征,最后将这些特征融合用于分割预测。
  • 对于监督学习,模型通过最小化LF和HF分割预测的监督损失进行优化。对于半监督学习,模型通过最小化标记图像的监督损失和未标记图像的LF和HF预测之间的一致性损失进行优化。
  • 基于小波的方法生成LF和HF图像用于一致性训练,有助于缓解之前半监督方法使用的人工扰动引起的学习偏差。
  • 在2D和3D生物医学数据集上的大量实验证明了所提出的XNet模型的有效性。
